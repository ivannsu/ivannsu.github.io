Menggunakan atau Menambahkan Fonts external ke dalam Website

Post Date: 30 Juni 2018

fonts external dapat di download di fonts.google.com setelah di download, silahkan extract. Hasil extract biasanya adalah file fonts dengan extensi berikut:

  • TrueType Fonts (.ttf)
  • OpenType Fonts (.otf)
  • The Web Open Font Format (.woff)
  • Embedded OpenType Fonts (.eot)
  • SVG Fonts (.svg) biasanya digunakan di icon seperti library fontAwesome
